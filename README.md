# TopsGarden - Modern Smart Garden System

**TopsGarden** is a personal project aimed at exploring the integration of IoT and automation in gardening. This project serves as a hands-on learning experience to develop a smart gardening system that simplifies plant care and water management.

### Project Goals

- Experiment with IoT technologies to automate gardening tasks.
- Build a user-friendly interface for monitoring and controlling garden conditions.
- Promote sustainable gardening practices through efficient resource usage.

### Key Features

- **Smart Watering System**: Automatically adjusts watering schedules based on environmental data.
- **Real-Time Monitoring**: Displays soil moisture, temperature, and humidity levels.
- **Customizable Settings**: Allows fine-tuning to suit different plant types and garden layouts.
- **Energy Efficiency**: Focuses on reducing water and energy consumption.

### Installation

1. Clone or download the repository to your local machine. Install the required dependencies:

    ```bash
    npm install
    ```

    Or, if using **Yarn**:

    ```bash
    yarn install
    ```

2. Start the development server to test and customize the project:

    ```bash
    npm run dev
    ```

    Or, with **Yarn**:

    ```bash
    yarn dev
    ```

This project is a work in progress and serves as a platform for learning and experimentation. Contributions and feedback are welcome!
